Authentication Security

MFA Implementation Checklist

✅ MFA Best Practices
  • Enforce MFA for all users - No exceptions, including executives
  • Prioritize phishing-resistant methods - FIDO2, Windows Hello, certificate-based
  • Block legacy authentication - POP, IMAP, SMTP Auth bypass MFA
  • Enable number matching - Prevents MFA fatigue attacks
  • Require MFA for privileged actions - Admin portals, sensitive data
  • Register multiple factors - Backup methods for account recovery

    KEY PRINCIPLES:
    • Higher tiers NEVER authenticate to lower tiers
    • Separate admin accounts per tier
    • Credential isolation between tiers

Break-Glass Account Security

⚠️ Emergency Access Accounts
  • Create minimum 2 break-glass accounts
  • Cloud-only accounts (not synced from AD)
  • Exclude from ALL Conditional Access policies
  • Use long, complex passwords (20+ characters)
  • Store credentials in physical safe or HSM
  • Alert on any sign-in attempt
  • Test quarterly, document in runbook

Monitoring & Detection

KQL - Identity Threat Detection Queries 
// Suspicious sign-in patterns - multiple failed then success
SigninLogs
| where TimeGenerated > ago(1h)
| summarize 
    FailedCount = countif(ResultType != "0"),
    SuccessCount = countif(ResultType == "0"),
    DistinctIPs = dcount(IPAddress)
    by UserPrincipalName
| where FailedCount > 5 and SuccessCount > 0
| project UserPrincipalName, FailedCount, SuccessCount, DistinctIPs

// Privileged role assignments
AuditLogs
| where OperationName has "Add member to role"
| where TargetResources has "Global Administrator" 
    or TargetResources has "Privileged Role Administrator"
| project TimeGenerated, InitiatedBy, TargetResources

// MFA method changes (potential account takeover)
AuditLogs
| where OperationName in ("User registered security info", 
    "User deleted security info", "User changed default security info")
| project TimeGenerated, UserPrincipalName = TargetResources[0].userPrincipalName,
    Operation = OperationName, InitiatedBy

// Impossible travel detection
SigninLogs
| where ResultType == "0"
| summarize 
    Locations = make_set(Location),
    Times = make_list(TimeGenerated)
    by UserPrincipalName, bin(TimeGenerated, 1h)
| where array_length(Locations) > 1
