IAM Terminology Quick Reference

Authentication (AuthN)
Verifying who someone is - proving identity
Authorization (AuthZ)
Determining what someone can access
SSO
Single Sign-On - one login for multiple apps
MFA
Multi-Factor Authentication - multiple proofs of identity
RBAC
Role-Based Access Control - permissions by job role
ABAC
Attribute-Based Access Control - dynamic permissions
Federation
Trust between identity providers across domains
IdP
Identity Provider - authenticates users
SP
Service Provider - the application being accessed
SAML
XML-based SSO protocol for enterprise apps
OAuth 2.0
Authorization framework for API access
OIDC
OpenID Connect - identity layer on OAuth 2.0
PIM
Privileged Identity Management - JIT admin access
PAM
Privileged Access Management - securing admin accounts
Zero Trust
Never trust, always verify - continuous validation
JIT
Just-In-Time access - temporary elevated permissions
