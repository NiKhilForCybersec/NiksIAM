SailPoint: Identity Governance Platform

ğŸ¯ What This Page Covers

SailPoint is the leader in Identity Governance and Administration (IGA). This covers IdentityNow (cloud) and IdentityIQ (on-prem), access certifications, provisioning, role management, and real enterprise use cases.

What is SailPoint?

SailPoint provides Identity Governance - ensuring the right people have the right access to the right resources. Unlike Okta (authentication/SSO), SailPoint focuses on access certification, provisioning automation, and compliance.

Automated Provisioning

SailPoint can automatically create/modify/delete accounts based on HR events (joiner/mover/leaver).

Joiner Provisioning Flow
TRIGGER: HR creates new employee in Workday

1. AGGREGATION (scheduled or real-time)
   SailPoint reads new employee from Workday:
   {
     "employeeId": "12345",
     "firstName": "Sarah",
     "lastName": "Chen",
     "department": "Engineering",
     "startDate": "2024-02-01"
   }

2. IDENTITY CREATION
   SailPoint creates Identity Cube for Sarah Chen
   Calculates which roles/access she should have (birthright)

3. BIRTHRIGHT PROVISIONING
   Based on rules:
   IF department = "Engineering" THEN:

5. VERIFICATION
   SailPoint aggregates from targets to confirm accounts created
   Updates Identity Cube with new accounts/entitlements

Connectors

SailPoint connects to systems via connectors (Virtual Appliance for on-prem, direct API for SaaS).

System TypeConnection MethodCapabilities
Active Directory VA + LDAP Full CRUD, group membership, password reset
Entra ID Graph API User management, group membership, licenses
Salesforce REST API User provisioning, profile/role assignment
ServiceNow REST API User/group management, role assignment
SAP VA + RFC/BAPI User management, role assignment
Databases VA + JDBC Read accounts, limited write
